# api & batch template

## build

```shell
docker-compose build
```

## run

batch

```shell
docker-compose -f docker-compose.batch.yml up
```

api

```shell
docker-compose -f docker-compose.api.yml up
```

open http://localhost:8080/docs on your browser.

## test

test

```shell
docker-compose -f docker-compose.test.yml up
```
